# High-Load Conversion UI — Кейс «Ремонт ПК»

## 🛠 О проекте
Коммерческий лендинг для высококонкурентной ниши (сервисный центр по ремонту компьютеров). Проект создавался в условиях «перегретого» рынка, где ключевыми факторами выживания были скорость загрузки, адаптивность и доверие пользователя.

**[🚀 Посмотреть Live Demo](https://xela07ax.github.io)**

---

### 🎯 На что обратить внимание:
*   **Фокус на главной:** Основная страница полностью отточена под мобильный трафик и готова к деплою.
*   **Скорость сборки:** Проект реализован в сжатые сроки для быстрой проверки бизнес-гипотез.
*   **Контент и доверие:** 100% уникальный текстовый контент. В визуалах участвует актер КВН из команды «Азия MIX» (фото предоставлено лично автором).

### ⚙️ Технические особенности (Legacy Stack):
*   **Frontend:** Bootstrap 3 (Customized), чистый HTML5/CSS3.
*   **Backend:** PHP-движок для мгновенной отправки SMS-уведомлений о заявках.
*   **Design:** Графика и макеты обработаны в Photoshop CC.
*   **Performance:** Оптимизация изображений и структуры для высокого рейтинга загрузки.

---

### 💡 Почему это в портфолио Go-разработчика?
Этот кейс демонстрирует мой инженерный подход к задачам:
1.  **Ориентация на результат:** Код пишется для решения конкретной бизнес-задачи.
2.  **Внимание к деталям:** Глубокая проработка UI/UX даже в простых формах.
3.  **Перфекционизм:** Стремление «вылизать» продукт под требования рынка, что я сейчас применяю при проектировании архитектуры и написании высокопроизводительных сервисов на **Go**.

> **Примечание:** Внутренние страницы находятся в состоянии *WIP (Work in Progress)*, так как основной упор делался на конверсию и запуск главного экрана.
